# TECH-STACK-001 — StackScore

**Vendor:** Bobkat IT  
**Category:** Technology Governance and Planning  
**Standard Status:** Preferred  
**Lifecycle Status:** Current  
**Stack Layer:** Strategy and Improvement

## Summary

StackScore is the strategic technology management platform used to assess environments, document maturity, identify risks, create recommendations, prioritize improvements, generate roadmaps, and communicate technology condition to business leadership.

## Primary Purpose

Translate technical findings into a measurable, prioritized technology improvement program.

## Why Bobkat IT Selected It

StackScore is purpose-built around the Bobkat IT consulting and managed service model. It connects assessments, recommendations, projects, roadmaps, reporting, and approved technologies in one platform.

## Business Value

- Provides leadership with a clear picture of technology condition
- Prioritizes the most important improvements
- Connects technical gaps to business risk
- Creates realistic implementation roadmaps
- Tracks improvement over time
- Supports repeatable consulting engagements
- Makes recommendations more transparent
- Improves accountability for technology projects

## Core Capabilities

- Technology maturity assessments
- Eight-pillar scoring model
- Category and overall health scores
- Historical score trends
- Rule-based recommendations
- Recommendation consolidation
- Executive reporting
- Roadmap creation
- Improvement plans
- Project creation
- Proposal and quote support
- Client lifecycle tracking
- Technology Catalog integration
- Playbook integration
- High-level environment reporting

## Standard Deployment

Each client organization should have:

- Organization record
- Primary contacts
- Assessment history
- Current maturity score
- Category scores
- Active recommendations
- Approved roadmap
- Related projects
- Technology Catalog mappings
- Review cadence
- Executive reporting access where included

## StackScore Pillar Coverage

StackScore owns and reports across all standard pillars:

1. Security and Risk
2. Identity and Access
3. Endpoint and Patch Management
4. Backup and Disaster Recovery
5. Network and Wi-Fi
6. Cloud and Collaboration
7. Data and Documentation
8. Monitoring and Visibility

## Common Recommendations

StackScore generates recommendations rather than serving as the implementation technology for every recommendation.

Examples:

- Replace unsupported firewall
- Deploy endpoint management
- Improve backup coverage
- Formalize identity lifecycle
- Implement network monitoring
- Standardize documentation
- Develop disaster recovery procedures
- Create a technology lifecycle roadmap
