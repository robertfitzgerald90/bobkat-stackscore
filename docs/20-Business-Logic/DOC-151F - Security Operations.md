# DOC-151F – Security Operations
## Purpose
This document defines the Security Operations assessment standard within the StackScore Technology Maturity Framework.
## Business Question
> Can the business detect, respond to, and reduce security risks?
## Desired Outcome
A mature security operations capability that supports secure, reliable business operations.
## Assessment Philosophy
Assess conversationally, remain vendor agnostic, and focus on business outcomes.
