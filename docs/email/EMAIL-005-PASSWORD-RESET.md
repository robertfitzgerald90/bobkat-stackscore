# EMAIL-005-PASSWORD-RESET.md

# Password Reset Email

## Purpose

Allow a customer to securely reset their StackScore password.

This email should be simple, security-focused, and avoid unnecessary marketing content.

---

# Trigger

Sent when a user requests a password reset from the login page.

---

# Subject

Reset Your StackScore Password

---

# Preview Text

A request was received to reset your StackScore password.

---

# Customer Journey

Forgot Password

↓

Receive Reset Email

↓

Reset Password

↓

Sign In

---

# Primary Goal

Allow the customer to securely reset their password.

---

# Required Components

- Bobkat IT Logo
- StackScore Logo
- Password Reset headline
- Reset Password button
- Security notice
- Footer

---

# Email Copy

## Hero Title

Reset Your Password

## Introduction

We received a request to reset the password for your StackScore account.

If you requested this change, click the button below to create a new password.

If you did not request a password reset, you can safely ignore this email. Your password will remain unchanged.

---

## Primary CTA

Reset My Password

---

# Security Notice

Your password reset link:

- Expires in 60 minutes
- Can only be used once

For your security, never share this link with anyone.

---

# Success Metric

Customer successfully resets their password and signs in.
