# Email Components

All StackScore emails should be assembled using reusable components.

## Components

### Email Layout

Shared wrapper used by every email.

---

### Header

Contains:

- Bobkat IT logo
- StackScore logo

---

### Hero

Contains:

- Email title
- Supporting description

---

### Primary Button

Large branded button.

Examples:

Activate Account

View Report

Reset Password

---

### Information Card

Reusable section for:

- Next steps
- Assessment details
- Proposal summary
- Project information

---

### Security Notice

Reusable component for:

- Expiration warnings
- One-time links
- Security reminders

---

### Footer

Contains:

Powered by Bobkat IT

Copyright

Website

Support email

Social links (future)
