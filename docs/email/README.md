# StackScore Email Documentation

## Purpose

This directory contains the standards for every email sent by StackScore.

All customer-facing emails should follow the same visual identity, branding, messaging, and component structure to provide a consistent and professional experience.

The goals of the email system are:

- Reinforce the StackScore brand
- Build trust with customers
- Provide clear next steps
- Maintain consistent communication
- Create reusable email templates
- Support future automation through Resend

This documentation is the source of truth for the StackScore email system.
