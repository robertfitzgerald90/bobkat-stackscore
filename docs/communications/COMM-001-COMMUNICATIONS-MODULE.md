# COMM-001 – Communications Module

## Vision
Build a centralized communications platform for StackScore that manages customer engagement from invitation through completed projects.

## Core Areas
- Overview Dashboard
- Templates
- Campaigns
- History
- Analytics
- Automations (future)

## Dashboard Metrics
- Emails Sent
- Delivered
- Open Rate
- Click Rate
- Invitations
- Activations
- Assessments Started
- Assessments Completed
- Roadmaps Viewed
- Proposals Viewed

## Functional Requirements
Administrators can preview, send, track and analyze communications.
Client users only see customer-facing events.
